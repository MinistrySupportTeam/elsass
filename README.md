# el-sass responsive grid builder

[el-sass](//github.com/NicolasGraph/el-sass) is a Sass powered CSS responsive and _semantic_ grids builder.  

## Features

* Unlimited columns width;
* unlimited custom breakpoints;
* highly customizable gutters;
* add-ons;
* etc.
