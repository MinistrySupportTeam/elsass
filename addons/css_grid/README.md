# Oui_grid_children

Extend oui_grid by adding a CSS grid.

* [Usage](#usage)
* [Settings](#settings)
* [Classes](#classes)
* [Credits](#credits)

## Usage

Import `_oui_grid_css_grid.scss` after `_oui_grid.scss` in your main `.scss` file.

```scss
@import "_oui_grid";
@import "_oui_grid_css_grid";
```
